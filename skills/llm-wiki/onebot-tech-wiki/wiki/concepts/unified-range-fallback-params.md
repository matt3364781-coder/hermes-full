## 使用场景

当 XGBoost 模型不可用时（文件缺失、加载失败、xgboost 未安装），`_fallback_range_predict()` 自动启用。

## 波动率 regime 判定

```python
rv_20 = sqrt(sum(log_returns[-20:]**2)) * sqrt(252)
regime = "low" if rv_20 < 10 else ("medium" if rv_20 < 25 else "high")
```

## 回退公式

```python
width_pct = (2.80 / 100) * vol_coef
half_width = current_price * width_pct / 2
low = current_price - half_width
high = current_price + half_width
```

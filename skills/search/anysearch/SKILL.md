# AnySearch

实时搜索引擎，支持通用搜索、垂直领域搜索（23 个领域）、并行批量搜索和 URL 内容提取。

触发条件：用户明确要求搜索时，考虑使用 AnySearch。

## CLI 命令

运行 `doc` 命令获取完整接口规范：
- Python: `python <skill_dir>/scripts/anysearch_cli.py doc`
- Node.js: `node <skill_dir>/scripts/anysearch_cli.js doc`
- Shell: `bash <skill_dir>/scripts/anysearch_cli.sh doc`

## API Key

匿名访问可用。如需更高频率，配置 `ANYSEARCH_API_KEY` 到 `.env`。

## 😤 Боль: поиск по экосистеме сломан

У Hermes Agent 111+ community-проектов. 264K+ звезд на GitHub. 12 категорий.

Но когда тебе нужно что-то конкретное — memory-провайдер, веб-интерфейс, шаблон деплоя — ты сталкиваешься с:

- **Разрозненными репозиториями** на десятках GitHub-аккаунтов
- **Нет единого индекса** — ты ищешь по issues, Discord, случайным блогам
- **Звезды врут** — проект на 5K звезд может быть заброшен, а на 100 звезд — именно то, что нужно
- **Неправильная категория** — ищешь "memory" и находишь 6 разных подходов без сравнения

Ты тратишь 20 минут на блуждание. Или сдаешься и пишешь с нуля.

Это не инженерия. Это археология.

## ✨ Решение: структурированная навигация

Этот skill превращает Hermes Atlas (https://hermesatlas.com) в searchable, filterable командный центр.

| Что нужно | Куда смотреть | Топовый выбор |
|-----------|---------------|---------------|
| **Персистентная память** | `/lists/best-memory-providers` | `mem0ai/mem0` — 54.6K звезд, официальный провайдер |
| **Переиспользуемые скиллы** | `/lists/top-skills` | `mukul975/Anthropic-Cybersecurity-Skills` — 754 скилла |
| **Веб-UI / GUI** | `/lists/workspaces-and-guis` | `nesquena/hermes-webui` — 5.3K звезд |
| **Docker / деплой** | `/lists/deployment-options` | `numtide/llm-agents.nix` — 1.1K звезд |
| **Multi-agent рой** | `/lists/multi-agent-frameworks` | `builderz-labs/mission-control` — 1.1K звезд |
| **Трекинг токенов** | `/lists/developer-tools` | `junhoyeo/tokscale` — 2.5K звезд |

### Ключевой инсайт

Hermes Atlas — это не просто список. Там есть:
- **Live-данные GitHub** — звезды, форки, недельный рост
- **Security review** — каждый проект проверен
- **6 курируемых списков** — мнение экспертов, с чего начать
- **12 категорий** — от Core до Domain Applications
- **Поиск** — клиентский фильтр по имени и описанию

**Этот skill заворачивает всё это в 5-шаговый workflow.**

## 🚀 Быстрый старт

### 1. Загрузи скилл

```bash
skill_view('hermes-atlas-navigator')
```

### 2. Найди ресурс

```bash
# Поиск по ключевому слову
"Мне нужна персистентная память для агента"
→ Перейти на /lists/best-memory-providers
→ Сравнить mem0 (54.6K), gbrain (12.7K), hindsight (11.8K)
→ Выбрать по критериям: managed vs self-hosted, граф vs вектор, локальный vs облако

# Или поиск по всей карте
"telegram bot ui"
→ Искать на hermesatlas.com по "telegram"
→ Найти hermes-telegram-miniapp (212 звезд), hermes-webui (5.3K)
→ Deep dive на странице проекта для фич, даты обновления, ссылки на GitHub
```

### 3. Проверь перед установкой

```bash
# Кликни "view on github" со страницы проекта в Atlas
# Проверь: качество README, последний коммит, открытые issues, лицензию
# Подтверди, что проект точно подходит под твою задачу
```

## 🛡 Подводные камни

| Симптом | Причина | Фикс |
|---------|---------|------|
| Поиск ничего не находит | Опечатка или слишком специфичный терм | Попробуй более общее слово или зайди в курируемый список |
| Много звезд, но проект мертв | Заброшен, но популярный | Проверь "last updated" на странице проекта |
| Скилл не устанавливается | Не совместим с agentskills.io | Проверь метку категории — некоторые репо это полноценные приложения, не скиллы |
| Нет горячего нового тулза | Atlas курируется еженедельно | Смотри на "+X/неделя" рост, не только общие звезды |
